🚀 ORC - INSTRUCCIONES RÁPIDAS
================================

📁 ARCHIVOS PRINCIPALES:
- streamlit_app.py    → Aplicación principal
- main_model.py       → Modelo físico
- requirements.txt    → Dependencias
- run_app.bat        → Ejecutar aplicación
- setup_env.bat      → Configurar entorno

⚡ INICIO RÁPIDO:
1. Doble clic en: setup_env.bat (solo la primera vez)
2. Doble clic en: run_app.bat (para ejecutar la app)

🔧 MANUAL:
1. Abrir terminal en esta carpeta
2. env\Scripts\activate
3. streamlit run streamlit_app.py

🌐 La aplicación se abre en: http://localhost:8501

📊 PESTAÑAS DE LA APLICACIÓN:
- ⚙️ Configuración de Torque
- 🚀 Condiciones Iniciales  
- ▶️ Simulación
- 📊 Resultados y Análisis

💡 CONSEJOS:
- Usa archivos Excel/CSV para cargar configuraciones
- Prueba diferentes funciones de torque (temporal/espacial)
- Exporta resultados en CSV/Excel
- Ajusta condiciones iniciales según tu caso

❓ PROBLEMAS:
- Si no funciona: ejecutar setup_env.bat nuevamente
- Verificar que Python esté instalado
- Usar navegador actualizado (Chrome/Firefox/Edge)
